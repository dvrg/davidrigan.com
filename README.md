# davidrigan.com

Personal website for David Rigan. Built with Next.js 15, React 19, Tailwind CSS v4, and Motion.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Based on Nim Template

This project uses the Nim template designed to combine minimalism with delightful animated components powered by [Motion-Primitives](https://motion-primitives.com).
